# Day 10 Lab Guide — Infrastructure as Code (IaC)
## Module 2 – Advanced CloudFormation Labs
### Date: 29-May-2026 (Friday)

# Lab Objectives

Participants will learn how to:

- Create advanced CloudFormation templates
- Build Nested Stacks
- Deploy StackSets
- Provision multi-environment infrastructure
- Store templates in S3
- Enable CloudTrail monitoring

# AWS Resources Used

- CloudFormation
- StackSets
- S3
- CloudTrail

# Lab Exercises

1. Create Advanced CloudFormation Stack
2. Build Nested Stacks
3. Multi-environment Provisioning
4. Deploy StackSets
5. Monitor with CloudTrail

# Sample Command

```bash
aws cloudformation validate-template \
--template-body file://advanced-stack.yaml
```
